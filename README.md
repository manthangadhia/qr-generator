This is a small tool written in JS that generates a QR-code given a URL as input.
This code is adapted for hosting online as a static site on github pages, as opposed to a small script I would run locally as a Python Notebook.

The front-end for this was generated (almost) entirely by Claude with the exception of font and some colour choices.

The fonts used have been acquired from Google Fonts, but they are hosted directly in the repository so that users are not being tracked by the google api each time they visit.
